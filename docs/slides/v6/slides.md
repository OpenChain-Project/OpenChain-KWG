# Phase 0 — 빌드 확인용 스켈레톤

이 슬라이드는 Slidev 빌드 인프라가 동작하는지 확인하기 위한 임시 슬라이드입니다.

- Phase 1: 디자인 토큰 + 컴포넌트 라이브러리
- Phase 2: 콘텐츠 기획 (격차 분석·비판·매핑·재구성안)
- Phase 3: 슬라이드 본문 재작성 (파트별)
- Phase 4: 페이싱·다이어그램·강사 노트
- Phase 5: 빌드·시각 검증
- Phase 6: 배포 전환

상세는 `SLIDES-WORK/PHASE.md` 참조.
