#### Part 1·Part 3 보충 설명

- **Part 1 표준 이해**: 개별 입증자료를 1:1 매핑하지 않고, `StandardCompare`(5230/18974/42001 비교)와 `HexCoreElements`(§3.1~3.6 6대 요소)로 입증자료 50개의 *전체 구조*를 조망하는 도입부로 사용. 각 입증자료의 상세는 Part 2-1~2-6에서 전개.
- **Part 3 AI 컴플라이언스**: ISO/IEC 42001은 입증자료 번호 체계가 없고 "체크포인트" 방식이므로 본 매핑(50개)에 포함되지 않음. `RegulatoryMatrix`·`LlamaChecklist`·`ML-BOM` 등 별도 컴포넌트로 다룸.
