## Marp 빌드 방법

### 사전 요구사항

Node.js 18+ 가 설치되어 있어야 합니다.

```bash
node --version   # v18 이상 확인
```

### 1회성 실행 (npx, 설치 불필요)

```bash
# PDF 변환 (권장 — 발표용)
npx @marp-team/marp-cli .claude/work/slides-v5.md --pdf -o slides-v5.pdf

# HTML 변환 (공유·웹 게시용)
npx @marp-team/marp-cli .claude/work/slides-v5.md -o slides-v5.html

# PowerPoint(PPTX) 변환
npx @marp-team/marp-cli .claude/work/slides-v5.md --pptx -o slides-v5.pptx
```

### 로컬 미리보기 (편집 시 실시간 확인)

```bash
npx @marp-team/marp-cli .claude/work/slides-v5.md --preview
# 브라우저에서 http://localhost:8080 열림, 파일 저장 시 자동 새로고침
```

### 전역 설치 후 사용 (반복 작업 시 더 빠름)

```bash
npm install -g @marp-team/marp-cli

# 이후 npx 없이 직접 실행 가능
marp .claude/work/slides-v5.md --pdf -o slides-v5.pdf
```

### VS Code에서 미리보기

1. VS Code 확장 **Marp for VS Code** 설치
2. `slides-v5.md` 파일 열기
3. 우측 상단 미리보기 아이콘 클릭 (또는 `Ctrl+Shift+P` → `Marp: Open Preview`)

### PDF 변환 시 한국어 폰트 깨짐 대응

PDF 변환 시 Noto Sans KR 폰트가 없으면 글자가 깨질 수 있습니다.

```bash
# Chrome/Chromium 기반 렌더링 사용 (권장)
npx @marp-team/marp-cli .claude/work/slides-v5.md --pdf \
  --pdf-notes \
  -o slides-v5.pdf

# 폰트 문제 발생 시: HTML로 먼저 변환 후 브라우저에서 PDF 인쇄
npx @marp-team/marp-cli .claude/work/slides-v5.md -o slides-v5.html
# → 브라우저에서 열기 → Ctrl+P → PDF로 저장
```
