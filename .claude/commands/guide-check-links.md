# /project:guide-check-links
# 가이드 내부 링크 유효성 점검

## 실행 순서

1. content/ko/guide/opensource_for_enterprise/ 하위 모든 파일을 읽는다
2. 파일 내 마크다운 링크([텍스트](경로)) 를 모두 추출한다
3. 각 링크의 대상 경로가 실제로 존재하는지 확인한다
4. 아래 형식으로 출력한다:

| 소스 파일 | 링크 텍스트 | 링크 경로 | 유효 여부 | 비고 |

유효 여부 기호: ✅ 유효 / ❌ 깨진 링크 / ⚠️ 구버전 경로

5. ❌ 와 ⚠️ 항목에 대해 올바른 경로를 제안한다
6. 승인 전까지 파일을 수정하지 않는다
